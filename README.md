# xcomment
将一些网站的评论区进行美化以方便更好地查看评论

### 支持的网站
v2ex.com

### 使用方式：
1. clone项目

2. 打开chrome系浏览器扩展管理 -> 加载解压缩的扩展 -> 选中刚刚 clone 的库的根目录

![](https://i.miji.bid/2024/01/25/5fd11fd84871a83271d1ce2884b59491.png)

3. 刷新评论区
